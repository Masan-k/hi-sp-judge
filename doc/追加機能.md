# 実装を後回しにする機能
* ★★★３Dパス（浮いたパス）　優先度：高　負荷：高
　→ゲーム性が上がるから普通の対戦に飽きたら絶対実装する
* ステージの作り込み　優先度：中　PG負荷：高
　→スマホのリリースするときにはほしい
* 味方のAI化　優先度：低　PG負荷：高
　→基本いらん。AIのお勉強には良いかも。
* ゲームパッド対応　優先度：低　PG負荷：高
　→スマホ対応後かな？スマホ版で遊べたほうが
　→キーボード操作がいまいちだから先かも…
* スマホ対応　優先度：中　PG負荷：高
　→それなりに作ってみて感触がよかったら…
* オンライン対戦　優先度：中　PG負荷：激高
　→ラグ問題が解決できる気がしない
